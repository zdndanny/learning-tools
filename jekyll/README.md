# Setup jekyll for writing blogs and static web sites

## setting up jekyll

Follow the instruction [for Ubuntu] (https://jekyllrb.com/docs/installation/#ubuntu)

``` 
$ sudo install ruby ruby-dev
$ mkdir ~/gems
$ gem install jekyll bundler
``` 

## write and preview blogs on local server

``` 
$ cd ~/workspace 
$ jekyll new myblog
$ cd myblog
$ jekyll serve
``` 
First, customize basic site infomation in '''_config.yml''' file.
